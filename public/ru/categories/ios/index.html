«Off-the-Record Messaging, обычно называемый OTR — криптографический протокол, предоставляющий сильное шифрование для мгновенных сообщений. OTR использует комбинацию симметричного шифрования AES, протокол обмена ключами Диффи — Хеллмана и хеш-функцию SHA-1. В дополнение к аудентификации и шифрованию, OTR предоставляет совершенную прямую секретность (perfect forward secrecy) и спорное шифрование (malleable encryption).

Основной идеей, стоящей за протоколом, было предоставление возможности отрицать для участников разговора, при этом сохраняя обмен сообщениями конфиденциальным, подобно приватным беседам в реальной жизни или разговор не под запись в журналистике.»

— Википедия

Примечания

Apple, Google и Microsoft часть проекта PRISM. Их проприетарные операционные системы не могут быть доверены для защиты вашей персональной информации от АНБ.

У нас есть две свободные альтернативы: GNU/Linux и BSD.

GNU/Linux имеет намного большее сообщество, которое может помочь вам с переходом. Рекомендуется начать с одного из дистрибутива GNU/Linux, который подойдет под ваши нужны.

Debian имеет давние традиции в области свободного программного обеспечения. Разработчики должны подписать социальный контракт и придерживаться этического манифеста. Строгие критерии включения ПО позволяют убедиться, что только сертифицированное открытое программное обеспечение будет поставляться в главном репозитории.

Gentoo является мета-дистрибутивом. Исходный код компилируется в бинарные приложения на пользовательском компьютере, что позволяет использовать практически безграничные возможности для адаптации и полную обратную прослеживаемость программной логики.

Linux Mint Debian Edition (LMDE) возможно самый простой для использования дистрибутив для людей, которые переходят с Microsoft Windows.

И Fedora, и openSUSE являются версиями для сообщества, которые служат стабильной основой для промышленных дистрибутивов GNU/Linux c коммерческой поддержкой. Компании по всему миру верят Red Hat Inc. и SUSE Linux GmbH из-за прозрачности всего процесса разработки.

Parabola GNU/Linux фактически на 99% состоит из Arch Linux с ядром, из которого удалены блобы и мета-пакетом, который блокирует несвободные лицензии. Это же можно и проделать с Arch.

Ubuntu от Canonical не рекомендуется PRISM Break, потому что имеет рекламу от Amazon и утечки информации по умолчанию. GNU/Linux дистрибутивы, основанные на Ubuntu, также пока не рекомендуются из-за некоторых других причин.

Примечания

Попробуйте использовать только Tor Browser Bundle (TBB) для просмотра веб-страниц. Он обеспечивает намного лучшую анонимность, чем любой другой веб-браузер. Убедитесь, что вы поняли основы Tor до того, как использовать его. Если сайт, который вы хотите посетить, не работает в TBB, попробуйте открыть его в Firefox, но осознайте, что этот браузер не анонимизирует ваш ip-адрес по умолчанию.

Советы по использованию TBB: Использование TBB для того, чтобы войти на вебсайты, которые содержат ваши реальные данные, бессмысленно, и это может заставить включиться защиту от мошенничества на сайте. Убедитесь, что открываете сайт через HTTPS до того, как войти на него через Tor. Вход на HTTP вебсайты может повлечь за собой захват ваших данных экзит-нодой Tor.

Советы по использованию Firefox: Этот браузер использует поиск Google по умолчанию: замените его более безопасной альтернативой. Другой альтернативой для Firefox является Iceweasel, браузер для дистрибутивов, основанных на Debian.
